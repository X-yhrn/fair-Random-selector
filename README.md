# 这是一个可以调整概率可以内定结果的很公平的随机选择器

一个为帮助用户在面临选择困难时做出决策的创意工具，结合随机性与个性化偏好设定，提升决策过程的趣味性和可视化体验。

## 🎯 项目特点

- ✨ **三种决策模式**：公平模式（等概率）、偏好模式（自定义权重）、命运模式（系统随机生成权重）
- 🌙 **主题切换**：支持白天/夜晚双主题，动态动画适配
- 📝 **选项管理**：添加、编辑、删除选项，支持Emoji图标和权重调节
- 🎲 **结果展示**：掷骰子动画后弹出结果模态框，附带趣味性描述
- 📊 **历史记录**：本地存储所有决策历史，支持按时间、收藏、使用频率分类查看
- 🔁 **一键复用**：从历史中快速加载已有选项与配置进行新决策
- 🔒 **数据安全**：所有数据仅存于你的设备上，保障隐私安全

## 🛠 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 7
- **样式方案**: TailwindCSS 3 + PostCSS + Autoprefixer
- **数据存储**: 浏览器LocalStorage API
- **部署平台**: 阿里云ESA Pages（边缘计算服务）


## 💡 使用指南

### 1. 添加选项
- 点击表情符号选择器，可以从预设的emoji中选择或自定义添加
- 输入选项描述，点击"添加选项"按钮
- 可以随时修改选项名称和emoji

### 2. 选择决策模式
- **公平模式**: 所有选项拥有完全相同的选中概率
- **偏好模式**: 通过滑块调整每个选项的权重，影响其被选中的概率
- **命运模式**: 系统随机分配权重，体验不可预测的命运

### 3. 开始决策
- 点击中间的大按钮进行决策
- 等待动画完成后查看结果
- 可以选择"再来一次"或"接受命运"

### 4. 查看历史记录
- 在历史记录面板中查看所有决策历史
- 可以按时间、收藏、常用等方式分类查看
- 支持复用历史决策配置

## 🎨 设计理念

本项目旨在解决日常生活中的选择困难问题，通过提供灵活的权重调节机制，让用户既能保持一定的随机性，又能根据自己的偏好影响决策结果。同时，通过趣味性的动画和交互设计，让决策过程更加轻松愉快。

## 📱 响应式设计

项目采用响应式设计，完美适配：
- 手机端（移动端优先）
- 平板设备
- 桌面端浏览器窗口

## 🔐 隐私政策

- 所有数据仅存储在用户的设备上
- 不上传任何数据到服务器
- 完全保护用户隐私
